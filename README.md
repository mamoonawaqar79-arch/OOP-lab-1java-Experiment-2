# Circle Area Calculator

## Description
A simple Java program that calculates the area of a circle using a predefined radius.

## Features
- Store circle radius
- Calculate area of the circle
- Display the result
- Uses classes and objects

## Concepts Used
- Classes and Objects
- Data Members
- Member Functions

## Sample Output
```text
Area = 78.5
```

## How to Run
```bash
javac Circle.java
java Circle
```
